There are three different kinds of operators: arithmetic, relational and conditional. Arithmetic operators and there expressions would be as follows: addition(+), subtraction(-), multiplication(*), division(/) and modulo(%) to find the remainder.
Then there is also post/pre increment and decrement, would are expressions that would be put in the code for the two functions to basically be carried out automatically.
An example of how this operator would look in coding would be x+n and for the post/pre increment would look like post increment(x++) pre increment(++x), the decrement expression would be shown the same way.
Relational operators are as follows: strictly less then(<), strictly les than or equal(<=), strictly greater than(>), strictly greater than or equal(>=), equal(==) and not equal(!=).
An example for this operator would look like in coding would be x<n.
And finally, conditional operators are as follows: logical and(&&), logical or(||) and logical not(!). One last example for how this operator would look in coding would be x||y.
