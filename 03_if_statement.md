An if statement takes a condition and runs it in the code to see if it is true or false.
An example would look like if(condition) 
                           {do something}
Another version of this statement is called an if-else if statment, where you account for two different conditions in one function.
The example is as follows if(condition)
                          {do something}
                          else if(condition2)
                          {do something else}
Another if statment is called and if-else if-else, where you account for two conditions and the last possible thing to do in the function.
The example is if(condition)
               {do something}
               else if(condition2)
               {do somehting else}
               else
               {do the last possible thing}
For the last if statment, it's much more straight forward and simple to do when you don't want to make a huge function.
The example is simply if(condition)
                      {do something}
                      else
                      {do another}
