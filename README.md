01_datatypes_variables.md

A datatype is basically a classification of what the information is for, so that the code can identify the variable.
There are many different datatypes,
For example, the number 4 would be an integer(int) or byte and the variable would be the name of what the integer is referring to, like currentTime.
When you are declaring a datatype, you could write it in two ways. Either int/byte = 4; or int/byte currentTime = 4;.
When writing the code, you can reference the variable by writing int or int currentTime, and the code will use the declaration from before as the value of the dataype and variable.
The reason datatypes and variables are used is so that when a code has a unknown value, it can be classified and used properly within the code.
