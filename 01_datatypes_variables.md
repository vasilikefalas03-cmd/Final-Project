A datatype is a declaration, allowing the variable to be reserved in memory, and the variable is the name of the declaration.
There are many different datatypes, such as Integer(int) or even known as Byte(byte) would be for whole numbers, Single(float) and Double(double) would be for deciaml/real numbers, Character(char) would be for single letters/digits/special characters or symbols, Boolean(bool) is for truth values, and String(string) is for a sequence of characters.
An example of a declaration, the number 4 would be an integer(int) and the variable would be the name of what the integer is referring to, like currentTime.
When you are declaring a datatype, you could write it in three ways.
Either int; which would be for the body of the code.
int = 4; 
or int currentTime = 4; 
Another example can be for char, i.e. char firstInitial = 'v'; and char firstInitial; in the code body.
When writing the char, you must put it between single quotes, always, or it'll be considered as an error.
When writing the code, you can reference the variable by writing int or int currentTime, and the code will use the declaration from before as the value of the variable.
The reason for datatypes and variables is so that it's easier to store in memory and ensure that the code runs smoothly.
